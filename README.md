# wordbox
一个背单词的程序，用`python`编写

## 使用方法
### 直接使用
打开[Releases](https://github.com/DannyWuzhiheng/wordbox/releases)

使用最新版本，目前：`beta1.0`

直接使用安装包
### 手动build
下载代码包，解压缩`books.zip`文件

```cmd
pip install pystray
pip install pillow
```
> [!TIP]
> 在国内请使用清华源加速
